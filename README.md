## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yash-xoxo/portfoliox.git
   cd portfoliox
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run locally:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
5. **Deploy to GitHub Pages:**
   ```bash
   npm run deploy
   ```
